To build and install this sample project.

If you don't already have it, you'll need to download aspectj from http://www.eclipse.org/aspectj/downloads.php.

1. Install the FoneMonkey console, which you can download from http://www.gorillalogic.com/fonemonkey4android/download.
2. Copy local.properties.template to local.properties. Edit and set sdk.dir to point to your android sdk.
3. Copy fonemonkey/fonemonkeybuild.properties.template to fonemonkey/fonemonkeybuild.properties. Edit and set aspectj.dir to point to your aspectj install directory.
4. Run ant debug install
5. Start the FoneMonkey Sample application on the emulator
5. Select Connect > Emulator from the FoneMonkey Console menu
6. Interact with the app and actions should record in the console

Learn more at http://www.gorillalogic.com/fonemonkey4android
